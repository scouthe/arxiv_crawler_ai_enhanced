**IEEE Transactions on Cloud Computing（TCC）** 是由 **IEEE Computer Society** 出版的国际学术期刊，主要面向 **云计算体系结构、平台系统、资源管理与云应用技术**等研究领域。

该期刊致力于推动 **云计算理论、系统架构与应用技术的发展**，汇聚来自分布式系统、数据中心计算、大规模数据处理与网络服务等方向的高水平研究成果，是云计算领域具有代表性的 IEEE Transactions 期刊之一。

TCC 的核心定位非常明确：**发表云计算相关算法、系统架构与应用实践方面的原创研究成果**，推动云平台与分布式计算系统的性能优化与技术创新。

在云计算研究体系中，IEEE Transactions on Cloud Computing 通常被视为 **云计算与分布式系统研究的重要学术期刊**，其整体定位大致相当于：

**云计算与分布式系统研究领域的重要 Transactions 期刊**

## **征稿范围**

**IEEE Transactions on Cloud Computing（TCC）** 主要关注 **云计算理论、系统与应用研究**，包括但不限于以下方向：

### **一、云计算体系结构与平台**

* 云数据中心体系结构
* 云平台与服务架构
* 虚拟化与容器技术

### **二、资源管理与调度**

* 云资源分配与负载均衡
* 弹性计算与自动扩展机制
* 数据中心能耗优化

### **三、云计算应用与服务**

* 大数据与云计算平台
* 移动云与边缘计算
* 云服务质量与性能评估

### **四、云计算安全与可靠性**

* 云环境数据安全与隐私保护
* 云系统容错与可靠性机制
* 云平台访问控制与安全策略

期刊强调 **理论方法与系统实现相结合**，鼓励研究成果在 **真实云平台或大规模实验环境**中进行验证。
